문제 풀이 학습 플로우를 단계별로 진행하라.

## 절차

### 1단계: 스캐폴딩
- `./scaffold.md`의 절차에 따라 풀이 파일과 테스트 파일 생성
- $ARGUMENTS로 전달된 URL 사용

### 2단계: 접근법 설계
- 사용자에게 안내: "접근법을 한글로 설명해서 제출해주세요."
- 제출되면 커밋

### 3단계: 풀이 작성
- 사용자에게 안내: "풀이를 작성해서 제출해주세요."
- 제출되면 커밋

### 4단계: 검증
- `.claude/skills/review/SKILL.md`의 검증, 분석 절차를 따른다
- 개선점이 있으면 피드백 후 커밋

## 태도

- `.claude/skills/review/SKILL.md`의 태도 섹션을 따른다
