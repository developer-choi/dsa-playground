## Video Ranking (비디오 랭킹)

### 문제 설명
비디오 배열을 받아서 **조회수(views)**를 기준으로 순위(Rank)를 매기는 함수를 작성해야 합니다.

**정렬 규칙:**
1. **조회수 내림차순:** 조회수가 높은 비디오가 1등입니다.
2. **원본 순서 유지 (Stable Sort):** 조회수가 같은 비디오끼리는 **입력 배열에 먼저 나온 비디오**가 더 높은 등수(작은 rank 번호)를 가집니다.

**반환값:**
`{ views, rank }` 형태의 객체를 담은 배열을 반환하되, 배열의 순서는 **원본 입력 배열의 순서(Original Order)** 와 동일해야 합니다. (랭킹 순으로 정렬해서 리턴하는 게 아닙니다)

### 예시
* **입력:** `views = [1000, 500, 750, 500]`
* **랭킹 산정:**
    1. `1000` (1등)
    2. `750` (2등)
    3. 첫 번째 `500` (3등) - 먼저 나왔으므로 상위 랭크
    4. 두 번째 `500` (4등)
* **출력:**
    * 인덱스 0: `{ views: 1000, rank: 1 }`
    * 인덱스 1: `{ views: 500, rank: 3 }`
    * 인덱스 2: `{ views: 750, rank: 2 }`
    * 인덱스 3: `{ views: 500, rank: 4 }`
