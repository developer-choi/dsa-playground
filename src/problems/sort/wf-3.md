# 등수 매기기

출처: 기업 코딩테스트

## 문제 설명

학생들의 점수 배열이 주어질 때, 각 학생의 등수를 반환한다. 등수는 자신보다 높은 점수를 가진 학생 수에 1을 더한 값이다. 동점자는 같은 등수를 가진다.

## 제한사항

- grade의 길이는 1 이상 1,000,000 이하
- grade의 각 원소는 1 이상 1,000,000,000 이하

## 입출력 예

| grade | result |
|-------|--------|
| [2, 2, 1] | [1, 1, 3] |
| [3, 2, 1, 2] | [1, 2, 4, 2] |

### 예제 설명

예제 1: 점수 2, 2, 1에서 — 2점은 자신보다 높은 점수가 없으므로 1등, 1점은 자신보다 높은 점수가 2명이므로 3등.

예제 2: 점수 3, 2, 1, 2에서 — 3점은 1등, 2점은 2등(2명이지만 같은 등수), 1점은 3명이 위에 있으므로 4등.
